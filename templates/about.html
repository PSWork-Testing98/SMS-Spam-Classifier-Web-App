System Overview

The Problem

SMS spam has become a major cybersecurity concern. Spam messages are often used for phishing attacks, fraud, and spreading malicious links. Detecting spam automatically helps users stay safe and avoid scams. Our system uses Natural Language Processing and Machine Learning to classify messages as spam or legitimate.

Technology Stack

  • Python — Core Programming Language
  • Pandas & NumPy — Data Processing & Analysis
  • NLTK — Text Preprocessing (Tokenization, Stopwords, Lemmatization)
  • Scikit-learn — Machine Learning Models & Evaluation
  • TF-IDF Vectorizer — Text Feature Extraction (Unigrams & Bigrams)
  • SMOTE (Imbalanced-learn) — Handling Class Imbalance
  • Multiple ML Models — SVM, Random Forest, KNN, Naive Bayes, Logistic Regression
  • Matplotlib & Seaborn — Data Visualization
  • Flask — Backend Web Framework (Model Deployment API)
  • Django — Backend Web Framework & Template Rendering
  • HTML, CSS — Frontend UI Design (Integrated with Django Templates)
  • Joblib — Saving & Loading Trained Model
